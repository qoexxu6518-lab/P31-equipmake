一、系统概述

装备制作系统是新增铁匠铺建筑的核心功能之一。玩家通过收集材料、消耗图纸，在铁匠铺内打造或强化各类装备，将装备佩戴给麾下武将以提升战力。

本系统包含三个子功能：装备打造、装备强化、装备分解。铁匠铺同一时间只能执行一个打造或强化任务，分解操作独立运行、不占用任务槽。

本版本大方向改动

  • 原兵装系统整体删除。原兵装系统所使用的背景图被复用为本装备制作系统的背景图。
  • 新增铁匠铺建筑，铁匠铺提供两个功能入口：「装备制作」（本文档所述内容）和「装备详情」（查看装备属性信息的独立界面，另行说明）。
  • 解锁方式调整：装备的打造/强化权限从绑定主公等级改为绑定铁匠铺等级，铁匠铺升级即可解锁更高品质的装备。

二、系统入口

入口路径：封地主界面 → 点击铁匠铺建筑 → 铁匠铺弹窗内点击「装备制作」按钮 → 进入装备制作主界面。

铁匠铺弹窗提供两个按钮：「装备制作」（进入本系统）和「装备详情」（查看当前装备信息，为独立功能，本文不作说明）。

三、铁匠铺建筑与升级

3.1 铁匠铺升级

铁匠铺本身可以升级。升级后可以解锁更高品质装备的打造和强化权限。各品质装备对应的铁匠铺解锁等级由配表配置，原型中以"铁匠铺 X 级解锁"标注。

3.2 升级方式

铁匠铺的升级方式（建设度/资源消耗/升级时间等）待定，另行说明。

3.3 未解锁装备的展示

当铁匠铺等级不足时，对应品质的装备在打造列表中以半透明 + 锁图标 + "铁匠铺 X 级解锁"文字展示，玩家可查看详情但无法发起打造或强化。

四、装备体系

4.1 装备分类

装备共分 5 个类型：武器、坐骑、令符、奇珍、宝典。每种类型有 5 个品质等级，共计 25 种装备。

📄 备注：每种类型各品质的具体装备名称及对应图标，将在独立的《装备名称与图标对照表》文档中提供，此处不作列举。

4.2 品质体系

品质由低到高依次为：白色 → 绿色 → 蓝色 → 紫色 → 橙色。品质决定装备的基础属性上限、刻蚀品质上限、洗炼词条上限，以及解锁所需的铁匠铺等级。低品质装备可通过"装备强化"功能升一级品质，每次只能升一级。

4.3 装备属性

每件装备拥有若干条基础属性（如攻击、防御、带兵等），属性条数和数值随品质提升而增加，均为固定值，非随机生成。高品质装备必然拥有低品质装备的全部属性，并在此基础上增加新属性或提升数值。

五、铁匠铺主界面

主界面是进入装备制作系统后的默认界面，根据铁匠铺当前状态分为三种显示模式。

5.1 空闲状态

  • 中央显示一个「+」加号按钮，点击后界面向左滑出，装备选择界面从右侧滑入。
  • 顶栏显示标题「装备制作」，右上角有「?」帮助按钮，点击打开帮助说明弹窗。
  • 底栏有返回按钮，点击返回封地主界面。

5.2 打造 / 强化进行中

  • 中央锻造圆环内显示目标装备图标（品质边框 + 类型图标）。圆环播放锻造动画：两把锤子交替敲击 + 多层火花粒子 + 撞击闪光 + 热力光晕 + 烟雾效果。
  • 装备名称显示于圆环正下方，属性信息在名称下方展示。强化任务时，属性区域显示属性对比（强化前 → 强化后）。
  • 再往下是进度条 + 剩余时间倒计时，实时更新。
  • 进度条下方两个操作按钮：
    • 「打造列表」/「强化列表」：点击后界面滑回装备选择界面，当前任务不中断、继续计时。
    • 「加速打造」/「加速强化」：调用游戏通用加速弹窗组件，玩家使用加速道具后任务立即完成，跳转至完成状态。

5.3 完成收取

  • 锻造圆环停止旋转，播放完成光效：中心爆发光圈扩散 + 向外放射 12 道光芒。
  • 界面显示「打造完成」或「强化完成」文字提示。
  • 显示「收 取」按钮。点击后触发通用道具获得界面（外部通用组件），关闭后主界面回到空闲状态，对应装备数量 +1。
  • 完成状态会一直保持直到玩家主动点击收取。玩家可离开后再回来，仍然看到待收取状态。
  • 完成状态下无法开始新的打造/强化，必须收取后才能再次使用铁匠铺。

六、装备打造

6.1 打造条件

  • 铁匠铺等级：部分高品质装备需要铁匠铺达到一定等级才能解锁；未解锁装备以半透明+锁图标+所需等级展示，不可发起打造。
  • 指定材料：每件装备的材料需求固定，材料不足时打造按钮不可点击。
  • 图纸：消耗 1 张该装备对应的图纸，图纸作为普通道具与其他材料一同在材料列表中展示。
  • 铁匠铺没有正在进行的任务；有任务时按钮置灰显示当前任务状态。

6.2 装备选择界面（打造模式）

  • 界面分为左侧装备列表和右侧装备详情两栏布局。
  • 底部导航栏：左侧返回箭头、中间「打造」/「强化」分页标签、右侧「装备分解」入口。
  • 顶部提供品质筛选和类型筛选，可组合使用，过滤列表显示的装备。

6.3 装备列表规则

列表展示全部 25 件可打造装备，排序逻辑（优先级由高到低）：

  1. 材料充足可立即打造：排最前，图标上显示绿色锤子标记。
  2. 已解锁但材料不足：正常显示，无特殊标记。
  3. 未解锁（铁匠铺等级不足）：半透明 + 锁图标 + 解锁所需等级文字。

同优先级内，按品质从高到低，再按类型顺序排序。装备图标左下角显示当前持有数量（数量≥1 时显示）。

6.4 装备详情面板

  • 装备品质边框 + 类型图标（大图）、装备名称
  • 全部基础属性列表
  • 刻蚀上限（该品质装备可容纳的刻蚀品质上限）
  • 洗炼上限（该品质可开启的洗炼词条条数）
  • 材料需求列表（含图纸），点击任意道具图标 → 调用通用道具弹窗（外部组件）
  • 预计打造时间
  • 「打造装备」按钮：确认后立即扣除材料，界面滑回主界面并进入打造进行中状态

6.5 打造完整流程

点击「打造装备」→ 扣除材料（含图纸）→ 主界面切换为进行中 → 倒计时结束 → 切换为完成状态 → 玩家点击「收取」→ 弹出通用道具获得界面 → 关闭后，装备进入背包（数量+1），主界面回到空闲状态。

七、装备强化

7.1 强化概念

装备强化的本质是将玩家已持有的一件低品质装备"升级"为高一级品质的同类型装备。强化时，消耗原装备 1 件，并消耗图纸和材料，完成后生成下一品质的同类装备。强化方向为品质直升：白→绿→蓝→紫→橙，每次只能升一个等级，橙色为最高品质不可继续强化。

7.2 强化条件

  • 背包中持有 1 件对应的低品质装备（强化开始时消耗）。
  • 对应的图纸 × 1（消耗）。
  • 所需的强化材料（种类和数量固定，通常多于打造同品质装备所需）。
  • 铁匠铺等级满足目标品质装备的解锁要求。
  • 铁匠铺当前无正在执行的任务。

7.3 强化列表规则

  • 强化列表展示玩家当前持有（count≥1）且存在更高品质版本的装备。
  • 列表显示的是当前持有的装备（不是目标装备），方便玩家找到手里有的装备。
  • 铁匠铺等级不足以解锁目标品质的强化项不出现在列表（不同于打造列表会显示锁定项）。
  • 若武将正在佩戴某件装备，该装备图标左上角显示武将头像标记。
  • 顶部提供「武将筛选」按钮，可过滤只显示某位武将装备中的装备。

7.4 强化详情面板

  • 品质图标对比：左侧当前品质图标 → 箭头 → 右侧目标品质图标
  • 装备名称对比：当前名称 → 强化后名称
  • 若武将正在佩戴该装备，显示武将头像 + 武将名称
  • 属性对比：每条属性显示「旧值 → 新值」，新值颜色高亮
  • 刻蚀：显示当前装备已有的刻蚀情况；强化后刻蚀不丢失，刻蚀品质上限随品质提升
  • 洗炼上限对比：旧上限条数 → 新上限条数
  • 材料需求（含图纸，不包含被消耗的低品质装备本身）
  • 预计强化时间
  • 「强化装备」按钮

7.5 强化后的装备归属

若被强化的装备正在被武将佩戴，强化完成并收取后，武将自动换装为强化后的新装备（佩戴关系保持，装备更新为高品质版本）。

7.6 强化与打造的共同规则

强化的整体流程与打造相同：确认→扣除材料→进行中→完成→收取。两者共用同一个铁匠铺任务槽，同时只能进行一种操作。

八、装备刻蚀（与本系统的交互部分）

刻蚀系统属于独立的子系统，本文档仅说明其与装备制作系统的交互关系。

8.1 刻蚀品质上限

每件装备有一个刻蚀品质上限，由装备品质决定。在装备打造的详情面板中，刻蚀行显示的是该装备可容纳的刻蚀品质上限（白色装备显示"白色"，绿色装备显示"绿色"，依此类推）。

8.2 强化时刻蚀的继承

装备强化时，原装备上已有的刻蚀自动转移到强化后的新装备上，不会丢失。同时，新装备的刻蚀品质上限随品质提升，之前受限的刻蚀词条可以进一步强化。

九、装备分解

9.1 分解入口

入口位于装备选择界面底部导航栏右侧「装备分解」按钮。分解功能不占用铁匠铺任务槽，可在打造/强化进行中时操作。

9.2 分解界面

  • 以网格形式展示所有可分解的装备：未被武将佩戴、无刻蚀、且持有数量≥1 的装备。有刻蚀的装备不可分解，不出现在分解列表中。
  • 顶部品质筛选下拉：可选择分解品质范围。
  • 玩家点击单件装备选中/取消选中，已选中装备高亮显示。
  • 「一键选中」按钮：按当前品质筛选，自动选中所有符合条件的装备，最多 15 件。
  • 右侧实时显示预计返还材料列表（根据选中装备累加计算）。

9.3 确认分解

  • 点击「分 解」按钮后，所有选中装备立即销毁，每件装备返还的材料由配表决定，返还材料自动入背包。
  • 被武将佩戴的装备、有刻蚀的装备均不出现在分解列表，无法被选中。
  • 分解操作不可撤销，执行前需玩家确认。

十、帮助弹窗

点击主界面或装备选择界面右上角「?」按钮，弹出帮助说明弹窗，覆盖当前界面。弹窗以纯文字形式说明本系统基本操作，供玩家快速查阅。弹窗可随时关闭，返回原界面，不影响当前任务状态。

十一、通用规则与限制

  • 铁匠铺同时只能进行一个打造或强化任务，不可并行。
  • 分解操作不占用任务槽，可在打造/强化期间操作。
  • 加速功能调用游戏通用加速弹窗组件（外部实现），加速后任务立即完成。
  • 材料/图纸道具点击调用通用道具弹窗组件（外部实现），不在本系统内实现。
  • 收取装备时调用通用道具获得界面（外部实现）。
  • 武将佩戴状态由武将系统维护，本系统仅读取「装备是否被佩戴」及「佩戴武将信息（头像/名称）」。
  • 所有打造/强化进度均为真实倒计时，玩家离线后继续计时，下次进入自动刷新状态。
  • 图纸作为普通消耗道具，由道具背包系统管理，打造/强化时扣除背包中对应图纸数量。
  • 装备持有数量无上限。
